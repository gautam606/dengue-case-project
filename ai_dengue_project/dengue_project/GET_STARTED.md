## 🚀 GET STARTED IN 3 STEPS

### Step 1: Install Dependencies (1 minute)
```bash
pip install -r requirements.txt
```

### Step 2: Train AI Models (2-3 minutes)
```bash
python train_models.py
```

**You'll see:**
```
OUTBREAK PREDICTION MODELS:
  Logistic Regression: 75-85%
  Random Forest (AI):  85-95%

PATIENT DIAGNOSIS MODELS:
  Logistic Regression: 80-85%
  Random Forest (AI):  85-90%
  SVM (AI):            82-88%

[SUCCESS] All models saved!
```

### Step 3: Run Web Application (instant)
```bash
cd app
python app.py
```

**Access:** http://127.0.0.1:5000
